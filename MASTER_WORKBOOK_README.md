## How to Update This File

### Adding Service Details (Texas/Florida properties)
1. Open Tab 2 (Service Details)
2. Find the property row (marked "TBD")
3. Replace TBD with actual info:
   - Container Type: Dumpster/Compactor/Cart
   - Container Size: X yards
   - Quantity: number of containers
   - Frequency: pickups per week
   - Total Capacity: total yards
4. Change Notes from "Service details needed" to "Confirmed"

### Updating Contract Info
1. Open Tab 4 (Contract Summary)
2. Find the property
3. Update:
   - Contract Status (Active/Expired/Review needed)
   - Notes (contract dates, terms, etc.)
